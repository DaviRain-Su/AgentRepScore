## Codebase Patterns
- **Script entry guard**: All new CLI scripts in `scripts/` should use `isMainModule()` (same pattern as `src/cli.ts` and `src/server.ts`) to prevent accidental execution during test imports.
- **Testnet RPC sensitivity**: `test/hardhat/integration.test.ts` runs against live X Layer testnet and will timeout/hang if the RPC is slow or unreachable. This can block `pnpm test:all` even when local/unit changes are correct.
- **Pool address configuration**: Uniswap V3 pool addresses are not hardcoded; they must be supplied via `UNISWAP_POOLS` env var as comma-separated addresses.
